## Avaliação de Suporte
Componente interativo de avaliação de suporte, onde o usuário seleciona uma nota de 1 a 5 e recebe uma tela de confirmação animada.
[Acesse Online](https://projeto-interativo-lk2w.vercel.app/)
## Tecnologias
React 19
Vite
Framer Motion
Lucide React

Como rodar localmente
bashgit clone https://github.com/gustavocncs/Projeto-Interativo-.git
cd Projeto-Interativo-
npm install
npm run dev
Acesse http://localhost:5173 no navegador.
